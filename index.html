Why we don't need y=mx+b

If anybody considers y = mx + b to be an equation, they simply are mistaken. It is clearly a function, in which an independent variable x is linked with a dependent variable y in such a way, that for every value of x, one and only one value of y will result. It becomes an equation if y is given a specific value, then x is not ‘independent’ anymore, it becomes the unknown variable, whose value to satisfy the equation will be found by solving the equation.
